# Seekey Wiki

欢迎！Seekey 是一个轻量的 Wayland 按键可视化工具——把最近几次按键以悬浮气泡的形式画在屏幕底部。

```
[ Ctrl + C ]  [ Ctrl + C x3 ]  [ hello world ]
```

**语言：** [English](Home) · **简体中文**

刚上手的话，最快的路径是：

```sh
make                       # 编译
./seekey --init-config     # 生成一个 ./seekey.ini 起点（可选）
./seekey --config-tui      # 在终端里改设置（可选）
./seekey                   # 运行
```

第一次运行没有气泡？你需要读权限访问 `/dev/input/event*`——`./install.sh` 会帮你配好。见 [[问题排查]]。
